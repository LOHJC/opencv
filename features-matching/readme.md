# features matching

## features
- scale invariant feature transform (SIFT)
    - 1 keypoint has a descriptor of 128 length

- oriented FAST and rotated BRIEF (ORB)
    - featues form accelerated segment test (FAST)
        - feature detector
    - binary robust independent elementart features (BRIEF)
        - feature descriptor

## matcher
- brute force (BF) matcher
- fast library for approximate nearest neighbor (FLANN) based matcher
